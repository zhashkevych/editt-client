# editt-frontend

## Project setup
```
npm install
```
You need to create .env.development and/or .env.production for requests to work, check out .env.example

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
